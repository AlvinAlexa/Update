Setup Guide
  1. Go to developer.twitter.com and create a Project + App.
  2. In App Settings → User authentication settings:
    • App type: Single Page App
    • Enable OAuth 2.0
    • Redirect URI: your GitHub Pages URL, e.g. https://user.github.io/repo/
      For local testing: http://localhost:8080
  3. Enable scopes: tweet.read tweet.write users.read offline.access
  4. Copy your Client ID (not Client Secret!) into the Configuration panel below.
  5. Set the Redirect URI field to exactly match your Twitter App setting.

Local testing: run npx serve . or python -m http.server 8080 in this file's directory.

Tokens are stored in localStorage in plain text. Use on trusted devices only.
