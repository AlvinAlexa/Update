Step 1 — CSS (one-time)

Go to Blogger → Theme → Edit HTML. Find ]]></b:skin> and paste the CSS from the CSS tab directly above it. Save.

Step 2 — Runtime Script (one-time)

The runtime script handles countdown timers, local time display, and LIVE/Ended status updates. Paste it once, just before </body> in your Blogger theme HTML — or include it in a custom HTML widget.

Step 3 — Card HTML (per post)

For each post, switch the Blogger editor to HTML mode and paste the card HTML. Each card has a data-kickoff UTC timestamp — the runtime script reads it and shows local time + live countdown automatically.

About data-kickoff

Times are stored in UTC ISO 8601 format (e.g. 2026-05-04T19:00:00Z). The runtime script converts to each visitor's local timezone automatically — no manual time conversion needed.
